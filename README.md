# CardWars
Mi juego consiste en una version simplificada del famoso juego de hearthstone o guerra de cartas de la famosa serie de hora de aventuras.
Los usuarios tendran un mazo de 4 cartas pero solo 1 estara en display es sera la que mostarra las stats cada carta con sus debidas cualidades.
Cada carta tendre un menu de acciones y debereran elegir bien cada 1 de ellas.
Gnara el primero que derrote 3 cartas del adversario.
